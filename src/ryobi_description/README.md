# ryobi_description


This package stores the URDF models, associated meshes, and sensory plugins forming the physical representation of the simulated agricultural robot.

> **Collision Physics**: In `ryobi_description`, the collision geometry is greatly simplified. Because the overarching aim of this project is fundamentally driven towards testing **image processing** and visual tracking algorithms rather than simulating accurate multi-body physics interactions, the environment and chassis definitions remain streamlined for computing efficiency!
